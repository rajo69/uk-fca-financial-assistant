# Regulatory Notes

This file is for the USER to fill in during Phase 3 from primary FCA / MaPS /
gov.uk sources. It is intentionally empty now. Notes must be in the user's
voice, not Claude Code's, because it represents the user's understanding of
the regulatory landscape — which is what the project demonstrates.

## Sources to read in Phase 3

- FCA PS25/22 — Targeted Support policy statement
- FCA Consumer Duty (PRIN 2A pages)
- FCA AGBR — Advice/Guidance Boundary Review documents
- FCA PS7/24 — Model Risk Management
- FCA Guidance for firms on the fair treatment of vulnerable customers
- MoneyHelper.org.uk content on pensions, ISAs, credit, insurance
- FCA Financial Services Register overview
- Gov.uk pages on personal savings allowance, pension allowances, ISAs

## Suggested note structure (delete if you prefer your own)

For each source: what it says, the linguistic patterns of compliant vs
non-compliant responses, the boundary conditions, the vulnerability signals
that trigger different routing.
